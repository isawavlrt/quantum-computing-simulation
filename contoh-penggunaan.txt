Eksperimen Quantum: Mini Quantum Lab

Eksperimen 1: Superposisi dengan Hadamard
1. Pilih **1 qubit**  
2. Aplikasikan **gate Hadamard (H)** pada `Q0`  
3. Lihat probabilitas **50%-50%** untuk `|0⟩` dan `|1⟩`  


Eksperimen 2: Entanglement Bell State
1. Pilih **2 qubit**  
2. Aplikasikan **Hadamard** pada `Q0`  
3. Aplikasikan **CNOT** dengan control `Q0` dan target `Q1`  
4. Lihat state **entangled**:  
   \[
   (|00⟩ + |11⟩)/√2
   \]

Eksperimen 3: Phase Manipulation
1. Aplikasikan **Hadamard → Pauli-Z → Hadamard** pada **1 qubit**  
2. Hasilnya setara dengan **Pauli-X!**

Pengembangan Lebih Lanjut
Aplikasi ini dapat dikembangkan dengan:

✨ Rotasi Gate (**Rx, Ry, Rz**) dengan parameter sudut  
✨ **Bloch Sphere Visualization** untuk single qubit  
✨ **Quantum Circuit Diagram** visual  
✨ **Predefined Algorithms** (Grover, Deutsch-Jozsa)  
✨ **Export ke Qiskit** untuk running pada real quantum hardware  
✨ **Noise Model** untuk simulasi lebih realistis
